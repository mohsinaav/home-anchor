# Assets Folder

This folder contains media assets for the Home Anchor landing page.

## Required Assets

### Demo Preview Video
- **File:** `demo-preview.mp4` (or `demo-preview.gif`)
- **Purpose:** Hero section animated preview showing the app in action
- **Recommended specs:**
  - Resolution: 1280x960 (4:3 aspect ratio)
  - Duration: 15-30 seconds loop
  - Format: MP4 (H.264) or GIF
  - Size: < 5MB for fast loading

### How to Create the Demo Video

1. **Open the app in demo mode:**
   - Go to `landing.html`
   - Click "Try Demo"
   - This loads sample data

2. **Record a screen capture showing:**
   - Navigating between family member tabs
   - The kids' points widget in action
   - Adding/completing activities
   - The family calendar
   - Switching between grid/focus view

3. **Tools for recording:**
   - macOS: QuickTime Player (Screen Recording)
   - Windows: Xbox Game Bar or OBS
   - Cross-platform: Loom, ScreenToGif

4. **Post-processing:**
   - Crop to 4:3 aspect ratio
   - Speed up to 1.5x if needed
   - Compress for web
   - Export as MP4 or convert to GIF
